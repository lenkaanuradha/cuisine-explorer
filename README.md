Cuisine Explorer is an LLM-powered application built with LangChain and Streamlit. Users can select a cuisine, and the app suggests popular restaurants and their signature dishes for the chosen cuisine.

Features:

LLM-based restaurant and dish recommendations using LangChain

Interactive frontend built with Streamlit

Simple input interface for selecting cuisine types

Returns curated suggestions for users looking for new dining experiences

Tech Stack:

Backend/Logic: Python, LangChain, OpenAI LLM

Frontend: Streamlit
